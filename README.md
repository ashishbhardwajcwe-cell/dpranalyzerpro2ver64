# dpranalyzerpro2ver63

DPR Analysis tool with all major IRC/MoRTH codes built in as guiding norms.

This is **Ver63**, an upgrade over Ver61 with the following additions:

1. **Clause-level citations** — every deficiency reports the source document, exact clause number, a one-line summary of what the clause requires, and how the DPR fails it.
2. **Three output formats** — PDF, editable Word (.docx), and a clean on-screen Print view.
3. **Guideline load status** — green banner on successful load, red blocking alert if any guideline fails; report footer lists the guideline versions referenced.
4. **Report header inputs** — Project Name, Road Number / Project Code, Reviewing Officer Name & Designation, and Date of Review (auto-filled with today). Populated into every report.
5. **Unsupported codes greyed out** — codes without a backing MD reference are disabled in the picker with a tooltip.
6. **Deeper analysis** — findings split into **Design Compliance** and **Documentation Compliance**, each with a **Recommended Corrective Action** and a **Severity Justification** tied to the cited clause.
